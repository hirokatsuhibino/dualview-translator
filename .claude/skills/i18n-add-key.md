# i18n全言語キー追加スキル

## 概要

`i18n.js` の翻訳辞書に新しいメッセージキーを追加する。
ユーザーからキー名と日本語・英語のテキストを受け取り、残り9言語を適切に翻訳して全11言語ブロックに挿入する。

## 手順

1. ユーザーから以下を受け取る:
   - キー名（例: `shortcutInfo`）
   - 日本語テキスト
   - 英語テキスト（任意。未指定なら日本語から推定）
   - 挿入位置のヒント（どのキーの後に追加するか、コメントセクション名等）

2. `i18n.js` を読み、各言語ブロックの該当位置を特定する

3. 以下の11言語分のテキストを生成:
   - `ja`: ユーザー指定
   - `en`: ユーザー指定 or 翻訳
   - `zh-CN`: 中国語簡体字
   - `zh-TW`: 中国語繁体字
   - `ko`: 韓国語
   - `fr`: フランス語
   - `de`: ドイツ語
   - `es`: スペイン語
   - `pt`: ポルトガル語
   - `ru`: ロシア語
   - `ar`: アラビア語

4. 各言語ブロックの同じ位置（指定されたキーの後）に `Edit` ツールで挿入

## ルール

- プレースホルダ `{param}` は全言語で維持すること
- HTMLタグ（`<strong>` 等）がある場合は全言語で同じ構造を維持
- 既存キーと重複しないことを確認
- 追加後、全11言語ブロックにキーが存在することを確認

## i18n.js の構造

```js
const MESSAGES = {
  ja: { key1: '...', key2: '...', ... },
  en: { key1: '...', key2: '...', ... },
  'zh-CN': { ... },
  'zh-TW': { ... },
  ko: { ... },
  fr: { ... },
  de: { ... },
  es: { ... },
  pt: { ... },
  ru: { ... },
  ar: { ... },
};
```

## 使用例

ユーザー: 「`autoTranslate` キーを追加して。日本語: '自動翻訳が有効です'」

→ 11言語ブロック全てに以下のようなキーを追加:
```js
// ja
autoTranslate: '自動翻訳が有効です',
// en
autoTranslate: 'Auto-translate is enabled',
// zh-CN
autoTranslate: '自动翻译已启用',
// ... 以下全言語
```
