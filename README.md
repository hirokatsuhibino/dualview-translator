# DualView Translator

原文と翻訳を**並べて表示**するブラウザ翻訳拡張。Chrome / Firefox 対応（Safari / iOS 対応は開発中）。

翻訳しても原文が消えないので、原文を確認しながら読めます。

## 特徴

- **デュアルビュー表示** — 原文の直下に翻訳を併記。原文を失わずに翻訳を確認
- **8つの翻訳モード** — 用途に応じて使い分け
- **自動翻訳ルール** — URLパターン＋CSSセレクタでサイトごとに自動翻訳を設定。SPA・動的コンテンツにも対応
- **翻訳リセット** — ×ボタンで要素ごとに個別リセット、またはポップアップからまとめてリセット
- **2つの翻訳エンジン** — Google 翻訳（無料）と DeepL（高品質）を切り替え
- **AI要約機能** — Claude / Gemini で翻訳結果を自動要約
- **翻訳結果キャッシュ** — 同じ文の再翻訳で API 呼び出しを省略し、速度向上 + API 使用量を節約
- **タブ式ポップアップ** — 翻訳操作と設定を分離したすっきりUI
- **キーボードショートカット** — よく使う操作をワンキーで実行
- **11言語対応** — 日英中韓仏独西葡露アラビア語に対応
- **ダーク/ライト テーマ** — ブラウザの設定に自動追従
- **多言語UI** — 拡張自体の表示言語を11言語から選択可能

## 翻訳モード

| モード | 操作方法 | 用途 |
|--------|---------|------|
| **選択翻訳** | テキストをドラッグ選択 | 気になる一文だけサッと翻訳 |
| **ページ全体翻訳** | ポップアップから実行 | 記事やドキュメントを丸ごと翻訳 |
| **ページ全体翻訳＆要約** | ポップアップから実行 | 翻訳＋AIによる全体要約 |
| **要素選択翻訳** | ポップアップから要素をクリック | ページの一部だけ翻訳 |
| **要素選択翻訳＆要約** | ポップアップから要素をクリック | ページの一部を翻訳＋AI要約 |
| **右クリック翻訳** | テキスト選択して右クリック | 選択テキストを即座に翻訳 |
| **要素翻訳** | 右クリック（テキスト未選択時） | クリックした場所のブロックを翻訳 |
| **要素翻訳＆要約** | 右クリック（テキスト未選択時） | クリックした場所を翻訳＋AI要約 |

さらに、外国語のページを開くと上部に翻訳バーが自動表示され、ワンクリックで「翻訳」または「翻訳＆要約」を実行できます。

## 自動翻訳ルール

特定のサイトを開いたとき、自動的に翻訳を実行するルールを登録できます。

### ルールの登録

1. ツールバーの拡張アイコンをクリック
2. 「ルール」タブを開く
3. URLパターン（例: `*://example.com/*`）を入力
4. 必要に応じてCSSセレクタで翻訳対象の要素を指定
   - **「要素を選択」ボタン**: ページ上の要素をクリックするだけでセレクタを自動生成
5. 翻訳モードを選択（翻訳のみ / 翻訳＆要約）
6. 「追加」をクリック

### ルールの編集

- 一覧の項目をクリックすると、下のフォームに値がセットされ**編集モード**に入ります
- URLパターン・セレクタ・モードを変更して「更新」で上書き
- 「キャンセル」で編集を取りやめ（追加モードに戻る）
- 同じ項目を再度クリックしても編集モードを解除できます

### 自動翻訳の動作

- **セレクタなし**: ページ全体を自動翻訳
- **セレクタあり**: 指定要素のみを翻訳し、コンテンツが書き換わると再翻訳（Webメール等に対応）
- **SPA対応**: URLが変わるたびにルールを再チェック

## 翻訳エンジン

| エンジン | APIキー | 特徴 |
|---------|--------|------|
| **Google 翻訳** | 不要 | 無料で即座に使える。デフォルト設定 |
| **DeepL** | 必要（無料登録可） | 高品質な翻訳。月50万文字まで無料 |

DeepL を使うには、[DeepL API](https://www.deepl.com/pro-api) で無料アカウントを作成し、ポップアップの設定でAPIキーを入力してください。

## AI要約機能

翻訳結果をAIで3〜5行に要約し、ページ上部に緑枠で表示します。

| エンジン | APIキー | 特徴 |
|---------|--------|------|
| **Claude** | 必要（有料） | Anthropic社のAI。高品質な要約 |
| **Gemini** | 必要（無料枠あり） | Google社のAI。無料で利用可能 |

ポップアップの「要約エンジン」からClaude / Geminiを選択し、APIキーを入力してください。
- Claude: [Anthropic Console](https://console.anthropic.com/) でAPIキーを取得
- Gemini: [Google AI Studio](https://aistudio.google.com/apikey) でAPIキーを取得

## キーボードショートカット

| キー | 操作 |
|------|------|
| `Ctrl+Shift+T` (Mac: `Cmd+Shift+T`) | ページ全体翻訳のトグル |
| `Ctrl+Shift+Y` (Mac: `Cmd+Shift+Y`) | 選択テキストを翻訳 |
| `Ctrl+Shift+R` (Mac: `Cmd+Shift+R`) | 領域選択モード |

ショートカットはブラウザの設定画面からカスタマイズできます:
- **Chrome / Edge**: `chrome://extensions/shortcuts`
- **Firefox**: `about:addons` → 歯車アイコン → 「拡張機能のショートカットキーの管理」

## インストール

### Chrome / Edge

1. [Releases](../../releases) から最新の ZIP をダウンロード
2. `chrome://extensions/` を開く
3. 右上の「デベロッパー モード」をオン
4. 「パッケージ化されていない拡張機能を読み込む」をクリック
5. 解凍した `dualview-translator` フォルダを選択

以下のサイト(Firefox Add-ons)から取得

https://addons.mozilla.org/en-US/firefox/addon/dualview-translator/

## 使い方

### 選択翻訳（最も手軽）

1. 翻訳したいテキストをマウスでドラッグ選択
2. フローティングパネルが自動表示される
3. 言語を選んで「翻訳する」をクリック
4. 原文と翻訳が並んで表示される

### ページ全体翻訳

1. ツールバーの拡張アイコンをクリック
2. 翻訳先言語を選択
3. 「ページ全体を翻訳」をクリック（要約付きは「ページ全体を翻訳＆要約」）
4. 各段落・見出しの下に翻訳が挿入される
5. 「翻訳をリセット」で元に戻す（個別リセットは各翻訳の「×」ボタン）

> スクロールで遅延読み込みされるコンテンツも自動的に翻訳されます。

### 要素選択翻訳

1. ツールバーの拡張アイコンをクリック
2. 「領域を選択して翻訳」をクリック（要約付きは「領域を選択して翻訳＆要約」）
3. ページ上の要素にカーソルを合わせるとオレンジ色のハイライトが表示される
4. 翻訳したい要素をクリックすると、その要素と子要素が翻訳される
5. Escキーでキャンセル

### 右クリック翻訳

- **テキスト選択中**: 右クリック → 「DualView: 翻訳」 → パネルに翻訳結果を表示
- **テキスト未選択**: 右クリック → 「DualView: この要素を翻訳」 → クリック位置のブロックをデュアルビュー翻訳
- **テキスト未選択**: 右クリック → 「DualView: この要素を翻訳＆要約」 → 翻訳＋AI要約を表示

### 翻訳エンジンの切り替え

1. ツールバーの拡張アイコンをクリック
2. 「設定」タブを開く
3. 「翻訳エンジン」で Google 翻訳 または DeepL を選択
4. DeepL を選んだ場合は API キーを入力

### 要約エンジンの設定

1. ツールバーの拡張アイコンをクリック
2. 「設定」タブを開く
3. 「要約エンジン」で Claude または Gemini を選択
4. APIキーを入力

### 表示言語の変更

ポップアップの「設定」タブ内にある「表示言語」から、拡張UIの表示言語を変更できます。

## 注意事項

- Google 翻訳の無料エンドポイントを使用しているため、大量リクエストで一時的にレート制限がかかる場合があります
- DeepL Free API の翻訳上限は月50万文字です
- 要約機能の利用にはClaude / Gemini のAPIキーが必要です
- `chrome://`、`about:` などのブラウザ内部ページでは動作しません

## プライバシー

- 翻訳テキストは選択した翻訳エンジン（Google / DeepL）のAPIにのみ送信されます
- 要約テキストは選択した要約エンジン（Claude / Gemini）のAPIにのみ送信されます
- APIキーはブラウザのローカルストレージにのみ保存され、外部に送信されません
- 翻訳履歴の収集やトラッキングは一切行いません
