# DualView Translator

原文と翻訳を**並べて表示**するブラウザ翻訳拡張。Chrome / Firefox 対応。

## 機能

| 機能 | 説明 |
|------|------|
| **選択翻訳** | ページ上でテキストを選択 → 翻訳パネルが自動表示。原文と翻訳を並べて確認。 |
| **ページ全体翻訳** | 全ての段落・見出しを翻訳し、原文の直下に翻訳を挿入（デュアルビュー）。 |
| **領域選択翻訳** | ドラッグで画面上の範囲を選択し、その中のテキストだけを翻訳。 |
| **翻訳エンジン** | Google 翻訳（無料・API キー不要）。 |
| **11言語対応** | 日本語, 英語, 中国語(簡/繁), 韓国語, フランス語, ドイツ語, スペイン語, ポルトガル語, ロシア語, アラビア語 |

## インストール方法

### Chrome / Edge

1. `chrome://extensions/` を開く
2. 右上の「デベロッパー モード」をオン
3. 「パッケージ化されていない拡張機能を読み込む」をクリック
4. `dualview-translator` フォルダを選択

### Firefox

1. `about:debugging` を開く
2. 「このFirefox」→「一時的なアドオンを読み込む」
3. `dualview-translator/manifest.json` を選択

> **Note**: 永続インストールは Firefox Add-ons (AMO) への登録が必要です。

## 使い方

### 選択翻訳（最も簡単）
1. 翻訳したいテキストをマウスでドラッグ選択
2. 自動的に翻訳パネルが表示される
3. 言語を選択して「翻訳する」ボタンをクリック
4. 原文と翻訳が並んで表示される

### ページ全体翻訳
1. 拡張機能アイコンをクリックしてポップアップを開く
2. 翻訳先言語を選択
3. 「ページ全体を翻訳」ボタンをクリック
4. 各段落の下に翻訳が挿入される
5. 「翻訳をリセット」で元に戻す

### 領域選択翻訳
1. ポップアップで「領域を選択して翻訳」をクリック
2. ポップアップが閉じ、カーソルが十字になる
3. 翻訳したい領域をドラッグ
4. 選択領域内のテキストのみ翻訳される
5. Esc でキャンセル

## 技術仕様

- **Manifest Version**: 3 (MV3)
- **翻訳 API**: `translate.googleapis.com` (Google Translate 非公式エンドポイント)
- **API キー**: 不要
- **権限**: `activeTab`, `storage`, `scripting`
- **同時翻訳数**: 最大 6 並列

## 注意事項

- Google 翻訳の無料エンドポイントを使用しているため、大量リクエストで一時的にレート制限がかかる場合があります
- `chrome://`, `about:` などのブラウザ内部ページでは動作しません
- Firefox では `about:debugging` から一時インストールのみ対応（開発・テスト用）
