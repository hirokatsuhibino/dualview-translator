# DualView Translator

日本語 | [English](README.en.md)

原文と翻訳を**並べて表示**するブラウザ翻訳拡張。Chrome / Firefox / iOS Safari / macOS Safari に対応。

翻訳しても原文が消えないので、原文を確認しながら読めます。

## 特徴

- **デュアルビュー表示** — 原文の直下に翻訳を併記。原文を失わずに翻訳を確認
- **8つの翻訳モード** — 用途に応じて使い分け
- **自動翻訳ルール** — URLパターン＋CSSセレクタでサイトごとに自動翻訳を設定。SPA・動的コンテンツにも対応
- **翻訳リセット** — ×ボタンで要素ごとに個別リセット、またはポップアップからまとめてリセット
- **3つの翻訳エンジン** — Google 翻訳（無料）／ DeepL（高品質）／ Apple 翻訳（オフライン・macOS Safari 限定）を切り替え。オフライン時は自動で Apple 翻訳にフォールバック
- **AI要約機能** — Claude / Gemini で翻訳結果を自動要約
- **音声読み上げ** — 訳文の各ブロックに 🔊 ボタンを配置。ブラウザ内蔵の音声合成で訳文を読み上げ（追加 API キー不要）
- **iOS / macOS Share Extension** — Safari 以外のアプリ（メモ・Mail・News など）からも共有シートで並列翻訳できる（v1.6 〜 / iOS 15+ / macOS 12+）
- **翻訳・要約キャッシュ** — 同じ文の再翻訳・再要約で API 呼び出しを省略し、速度向上 + コスト削減。ヒット率を設定タブに表示
- **タブ式ポップアップ** — 翻訳操作と設定を分離したすっきりUI
- **キーボードショートカット** — よく使う操作をワンキーで実行
- **11言語対応** — 日英中韓仏独西葡露アラビア語に対応
- **ダーク/ライト テーマ** — ブラウザの設定に自動追従
- **多言語UI** — 拡張自体の表示言語を11言語から選択可能

## 翻訳モード

| モード | 操作方法 | 用途 |
|--------|---------|------|
| **選択翻訳** | テキストをドラッグ選択 → 表示されるアイコンをクリック | 気になる一文だけサッと翻訳 |
| **ページ全体翻訳** | ポップアップから実行 | 記事やドキュメントを丸ごと翻訳 |
| **ページ全体翻訳＆要約** | ポップアップから実行 | 翻訳＋AIによる全体要約 |
| **要素選択翻訳** | ポップアップから要素をクリック | ページの一部だけ翻訳 |
| **要素選択翻訳＆要約** | ポップアップから要素をクリック | ページの一部を翻訳＋AI要約 |
| **右クリック翻訳** | テキスト選択して右クリック | 選択テキストを即座に翻訳 |
| **要素翻訳** | 右クリック（テキスト未選択時） | クリックした場所のブロックを翻訳 |
| **要素翻訳＆要約** | 右クリック（テキスト未選択時） | クリックした場所を翻訳＋AI要約 |

さらに、外国語のページを開くと上部に翻訳バーが自動表示され、ワンクリックで「翻訳」または「翻訳＆要約」を実行できます。

## 自動翻訳ルール

特定のサイトを開いたとき、自動的に翻訳を実行するルールを登録できます。

### ルールの登録

1. ツールバーの拡張アイコンをクリック
2. 「ルール」タブを開く
3. URLパターン（例: `*://example.com/*`）を入力
4. 必要に応じてCSSセレクタで翻訳対象の要素を指定
   - **「要素を選択」ボタン**: ページ上の要素をクリックするだけでセレクタを自動生成
5. 翻訳モードを選択（翻訳のみ / 翻訳＆要約）
6. 「追加」をクリック

### ルールの編集

- 一覧の項目をクリックすると、下のフォームに値がセットされ**編集モード**に入ります
- URLパターン・セレクタ・モードを変更して「更新」で上書き
- 「キャンセル」で編集を取りやめ（追加モードに戻る）
- 同じ項目を再度クリックしても編集モードを解除できます

### 自動翻訳の動作

- **セレクタなし**: ページ全体を自動翻訳
- **セレクタあり**: 指定要素のみを翻訳し、コンテンツが書き換わると再翻訳（Webメール等に対応）
- **SPA対応**: URLが変わるたびにルールを再チェック

## 翻訳エンジン

| エンジン | APIキー | 特徴 |
|---------|--------|------|
| **Google 翻訳** | 不要 | 無料で即座に使える。デフォルト設定 |
| **DeepL** | 必要（無料登録可） | 高品質な翻訳。月50万文字まで無料 |
| **Apple 翻訳** | 不要 | macOS Safari 限定。オンデバイスでネットワーク不要・APIキー不要 |

DeepL を使うには、[DeepL API](https://www.deepl.com/pro-api) で無料アカウントを作成し、ポップアップの設定でAPIキーを入力してください。

Apple 翻訳は macOS Safari 環境のみで利用可能（拡張起動時に自動検出されたとき設定タブに選択肢が表示されます）。Google / DeepL を選択中でもネットワーク不通時は自動で Apple 翻訳にフォールバックします（cache hit がある場合は cache を優先）。

## AI要約機能

翻訳結果をAIで3〜5行に要約し、ページ上部に緑枠で表示します。

| エンジン | APIキー | 特徴 |
|---------|--------|------|
| **Claude** | 必要（有料） | Anthropic社のAI。高品質な要約 |
| **Gemini** | 必要（無料枠あり） | Google社のAI。無料で利用可能 |

ポップアップの「要約エンジン」からClaude / Geminiを選択し、APIキーを入力してください。
- Claude: [Anthropic Console](https://console.anthropic.com/) でAPIキーを取得
- Gemini: [Google AI Studio](https://aistudio.google.com/apikey) でAPIキーを取得

## 音声読み上げ

各訳文ブロックの右側に 🔊 ボタンが表示されます。クリックすると翻訳先言語の音声で訳文を読み上げます。

| 配置 | 表示 |
|------|------|
| 選択翻訳パネル | コピーボタンの隣に常時表示 |
| インライン翻訳ブロック | × の左、ホバー時のみ表示 |
| 要約ブロック | 「要約」バッジの隣に常時表示 |

- 再生中はアイコンが ⏹ に切り替わり、再クリックで停止
- 別のボタンを押すと現在の再生を止めて新規再生
- `Esc` キー、別タブへ切り替え、ページ離脱でも自動停止
- 使用するのはブラウザ内蔵の Web Speech API のため**追加の API キーや権限は不要**
- 対象言語の音声が OS にインストールされていない場合（例: Windows でアラビア語音声未導入）はトーストで通知し再生は行わない

## Share Extension（iOS / macOS）

v1.6 から、Safari 以外のアプリのテキストも共有シート経由で翻訳できます。

| 操作 | 動作 |
|------|------|
| メモ / Mail / News 等でテキスト選択 → 共有 → **DualView Translator** | 並列翻訳ビューがシート / フローティングウィンドウで開く |
| 翻訳ビューの **コピー** | 訳文をクリップボードにコピー（2 秒間「✓ コピー済」表示） |
| 翻訳ビューの **閉じる** / `Esc`（macOS） | Extension を閉じる |

- 対応 OS: **iOS 15.0+ / macOS 12.0+**（既存の Safari Web Extension は引き続き macOS 10.14+ で動作）
- 翻訳エンジン: **Google Translate**（v1）。要約 / DeepL / Apple 翻訳は v2 以降で順次追加予定
- 翻訳先言語などの設定は Web 拡張で変更したものを App Group 経由で自動共有（再設定不要）
- 共有時は受け取ったテキストを GET URL に乗せる仕様だが、`URLSession.ephemeral` + `URLCache` 無効化により**端末内のディスクキャッシュ**にはテキストを残さない（Google 側のサーバーアクセスログや中間プロキシなど経路上のログには記録され得る点はご留意ください）

App Store 配布版に同梱しています（個別の有効化操作は不要・共有シートの App リストに自動で表示）。

## 使い方

### 選択翻訳（最も手軽）

1. 翻訳したいテキストをマウスでドラッグ選択
2. 選択範囲の右下隣に小さな翻訳アイコンが表示される
3. アイコンをクリックするとフローティングパネルが開く
4. 言語を選んで「翻訳する」をクリック
5. 原文と翻訳が並んで表示される

### ページ全体翻訳

1. ツールバーの拡張アイコンをクリック
2. 翻訳先言語を選択
3. 「ページ全体を翻訳」をクリック（要約付きは「ページ全体を翻訳＆要約」）
4. 各段落・見出しの下に翻訳が挿入される
5. 「翻訳をリセット」で元に戻す（個別リセットは各翻訳の「×」ボタン）

> スクロールで遅延読み込みされるコンテンツも自動的に翻訳されます。

### 要素選択翻訳

1. ツールバーの拡張アイコンをクリック
2. 「領域を選択して翻訳」をクリック（要約付きは「領域を選択して翻訳＆要約」）
3. ページ上の要素にカーソルを合わせるとオレンジ色のハイライトが表示される
4. 翻訳したい要素をクリックすると、その要素と子要素が翻訳される
5. Escキーでキャンセル

### 右クリック翻訳

- **テキスト選択中**: 右クリック → 「DualView: 翻訳」 → パネルに翻訳結果を表示
- **テキスト未選択**: 右クリック → 「DualView: この要素を翻訳」 → クリック位置のブロックをデュアルビュー翻訳
- **テキスト未選択**: 右クリック → 「DualView: この要素を翻訳＆要約」 → 翻訳＋AI要約を表示

### 翻訳エンジンの切り替え

1. ツールバーの拡張アイコンをクリック
2. 「設定」タブを開く
3. 「翻訳エンジン」で Google 翻訳 または DeepL を選択
4. DeepL を選んだ場合は API キーを入力

### 要約エンジンの設定

1. ツールバーの拡張アイコンをクリック
2. 「設定」タブを開く
3. 「要約エンジン」で Claude または Gemini を選択
4. APIキーを入力

### 表示言語の変更

ポップアップの「設定」タブ内にある「表示言語」から、拡張UIの表示言語を変更できます。

### 端末間で設定を同期

ポップアップの「設定」タブ内「端末間で同期」セクションのチェックボックスを ON にすると、複数の端末で設定が自動同期されます。

- 同期対象: 翻訳先言語、翻訳エンジン、要約エンジン、自動翻訳ルール、除外ドメイン、UI 表示言語
- **APIキー**: 同期されません（クラウド経由の漏洩を防ぐため）。各端末で個別に設定してください
- 同期エンジン: Chrome / Edge は Google アカウント、Firefox は Firefox アカウント、Safari は iCloud Keychain（ベータ扱い）
- 初回 ON 時はクラウド優先で合流（他端末で既に同期されている設定を尊重）
- 容量制限（chrome.storage.sync の 8KB/item 制限）を超えた場合はエラー表示。自動翻訳ルールが多すぎる場合に発生し得ます

### 設定のバックアップ（エクスポート / インポート）

ポップアップの「設定」タブ内「設定のバックアップ」セクションから、現在の設定を JSON ファイルに書き出したり、別の端末や以前のバックアップから読み込んだりできます。

- 対象: 翻訳先言語、翻訳エンジン、要約エンジン、自動翻訳ルール、除外ドメイン、UI 表示言語
- **テーマ**: バックアップ対象外（OSの `prefers-color-scheme` から毎ページ再判定されるため、インポートしても上書きされる）
- **APIキー**: 既定で **含まれません**。「APIキーを含める」チェックボックスを ON にしたときだけ書き出されます（漏洩リスク回避のため明示的に opt-in）
- 翻訳・要約キャッシュおよびヒット率統計はバックアップ対象外（端末固有のため）
- インポート方式: Chrome / Edge / Safari では「ファイルから読み込む」ボタンで `.json` を直接選択できる。Firefox はブラウザ仕様上 popup が閉じてしまうため、ファイルをテキストエディタで開いて内容をコピーし、テキストエリアに貼り付ける必要がある（ボタンは Firefox では非表示）
- インポートは **マージ動作**: JSON に含まれるキーのみが書き換わり、含まれていないキー（例: opt-in OFF でエクスポートしたときの API キー）はインポート後も既存値を維持します
- バージョン非互換の JSON（将来 v2 形式など）は安全のため拒否されます

## 注意事項

- Google 翻訳の無料エンドポイントを使用しているため、大量リクエストで一時的にレート制限がかかる場合があります
- DeepL Free API の翻訳上限は月50万文字です
- 要約機能の利用にはClaude / Gemini のAPIキーが必要です
- `chrome://`、`about:` などのブラウザ内部ページでは動作しません

## プライバシー

- 翻訳テキストは選択した翻訳エンジン（Google / DeepL）のAPIにのみ送信されます
- 要約テキストは選択した要約エンジン（Claude / Gemini）のAPIにのみ送信されます
- APIキーはブラウザのローカルストレージにのみ保存され、外部に送信されません
- 翻訳履歴の収集やトラッキングは一切行いません
