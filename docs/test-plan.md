# DualView Translator テストプラン

## テスト環境

- **ブラウザ**: Chrome（最新）、Firefox（112+）、Edge（最新）、Safari（macOS 16.4+ / iOS 15+）
- **OS**: macOS / Windows / Linux
- **テーマ**: ダークモード / ライトモード
- **テスト用ページ例**:
  - 英語記事: https://en.wikipedia.org/wiki/Main_Page
  - 中国語: https://zh.wikipedia.org/
  - Lazy load: https://news.ycombinator.com/ (コメント展開)、Twitter/X（infinite scroll）
  - SPA: GitHub Issues ページ（ページ遷移なしでコンテンツ変化）
