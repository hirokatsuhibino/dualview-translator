# Chrome Web Store 掲載用テキスト

## 拡張機能名

DualView Translator

## 簡単な説明（132文字以内）

原文と翻訳を並べて表示する翻訳拡張。翻訳しても原文が消えないので、文脈を確認しながら読めます。Google翻訳/DeepL対応、AI要約機能搭載。

## 詳細説明

DualView Translatorは、原文と翻訳を並べて表示するブラウザ翻訳拡張です。

翻訳しても原文が残るため、文脈を確認しながら自然に外国語の記事やドキュメントを読み進められます。

■ 主な特徴

・デュアルビュー表示 — 原文の直下に翻訳を併記
・8つの翻訳モード — 選択翻訳、ページ全体翻訳、要素選択翻訳＆要約など
・自動翻訳ルール — URLパターンで特定サイトを自動翻訳。CSSセレクタで対象要素を絞り込み可能
・翻訳リセット — ×ボタンで要素ごとに個別リセット、またはまとめてリセット
・2つの翻訳エンジン — Google翻訳（無料）とDeepL（高品質）
・AI要約機能 — Claude / Geminiで翻訳結果を3〜5行に要約
・タブ式ポップアップ — 翻訳操作と設定を分離したすっきりUI
・キーボードショートカット — Ctrl+Shift+Tでページ翻訳など
・11言語対応UI — 日英中韓仏独西葡露アラビア語
・ダーク/ライトテーマ — OSの設定に自動追従

■ 翻訳モード

1. 選択翻訳 — テキストを選択するだけで翻訳パネルが表示
2. ページ全体翻訳 — 記事やドキュメントを丸ごとデュアルビュー翻訳
3. ページ全体翻訳＆要約 — 翻訳に加えてAIによる全体要約を表示
4. 要素選択翻訳 — ページの特定の要素をクリックして翻訳
5. 要素選択翻訳＆要約 — 要素をクリックして翻訳＋AI要約
6. 右クリック翻訳 — テキスト選択後、右クリックで即座に翻訳
7. 要素翻訳 — 右クリックでブロック単位の翻訳
8. 要素翻訳＆要約 — 右クリックで翻訳＋AI要約

■ 翻訳バー

外国語のページを開くと画面上部に翻訳バーが自動表示されます。ワンクリックで「翻訳」または「翻訳＆要約」を実行できます。

■ 自動翻訳ルール

よく使うサイトをルール登録しておくと、開くだけで自動翻訳が実行されます。URLパターン＋CSSセレクタで対象を細かく指定でき、ページ上の要素をクリックするだけでセレクタを自動生成する要素ピッカー機能も搭載。SPA・動的コンテンツ・Webメール等にも対応しています。

■ 動的コンテンツ対応

スクロールで遅延読み込み（lazy load / infinite scroll）されるコンテンツも自動的に検出して翻訳します。

■ 翻訳エンジン

・Google翻訳: APIキー不要、無料で即座に利用可能（デフォルト）
・DeepL: 高品質な翻訳。無料APIキーで月50万文字まで利用可能

■ AI要約機能

翻訳結果をClaude（Anthropic）またはGemini（Google）のAIで要約します。APIキーの設定が必要です。

■ プライバシー

・翻訳テキストは選択した翻訳エンジンのAPIにのみ送信
・APIキーはブラウザのローカルストレージにのみ保存
・翻訳履歴の収集やトラッキングは一切行いません

■ キーボードショートカット

・Ctrl+Shift+T: ページ全体翻訳
・Ctrl+Shift+Y: 選択テキスト翻訳
・Ctrl+Shift+R: 要素選択モード

## 言語

## スクリーンショット説明文

### スクリーンショット1: デュアルビュー表示
原文の直下に翻訳を並べて表示。文脈を確認しながら読めます。

### スクリーンショット2: 選択翻訳
テキストを選択するだけでフローティングパネルが表示され、即座に翻訳できます。

### スクリーンショット3: タブ式ポップアップ
「翻訳」タブで素早く操作、「設定」タブでエンジンやAPIキーを管理。

### スクリーンショット4: AI要約機能
翻訳結果をAIが3〜5行に要約。ページの内容を素早く把握できます。

### スクリーンショット5: 翻訳バー
外国語ページを開くと自動で表示。ワンクリックで翻訳開始。
